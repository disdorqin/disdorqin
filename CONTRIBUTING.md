# Contributing to disdorqin Profile

This repository contains my GitHub profile README and related assets.

## How to Suggest Changes

1. **Open an issue** describing your suggestion (new badge, link fix, content improvement)
2. **Create a pull request** if you have the implementation ready

## Guidelines

- Keep the design consistent with the existing neon/cyber theme
- Test that all external image links and badges render correctly
- Do not include personal contact information beyond what's already public
- SVG files in `assets/` should be lightweight and scalable

## Content Scope

This profile focuses on:
- Time Series Forecasting
- Power Markets & Energy Systems
- Automated Research Tools
- AI Literature Mining

Suggestions related to these topics are most relevant.
